# 🚀 Playwright con Cucumber (BDD)

Este proyecto tiene como objetivo establecer una base sólida para implementar automatización de pruebas utilizando **Playwright + Cucumber + TypeScript**, siguiendo el enfoque BDD (Behavior Driven Development).

El propósito es contar con una arquitectura simple pero escalable, que permita construir pruebas E2E mantenibles y reutilizables.

📌 Desarrollado por: **Aldo Ernesto Salazar Lavado – QA Architect**

---

# ✅ Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

- Node.js
- Visual Studio Code / IntelliJ / WebStorm / Cursor

### Extensiones recomendadas (VS Code)

- Cucumber (Gherkin)
- Playwright
